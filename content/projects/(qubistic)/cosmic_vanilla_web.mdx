# CosmicVanilla Web

CosmicVanilla Web is a custom website designed specifically for the Cosmic Vanilla Minecraft server. It serves as a hub for the server's information, community updates, and provides an immersive experience for players and visitors.

## Features

- **Interactive Design**: The website is built using React and Next.js, offering a dynamic and interactive user interface.

- **Performance and SEO**: Next.js enables server-side rendering, enhancing website performance and improving search engine optimization (SEO) for better discoverability.

- **GitHub Integration**: The project is hosted on GitHub, allowing for version control and collaboration with other developers.

- **Deployment**: The website is deployed on Vercel, ensuring fast and reliable hosting with automatic scaling and continuous deployment.

## Technologies Used

- **Language**: TypeScript
- **Framework**: React, Next.js
- **Version Control**: GitHub
- **Hosting**: Vercel
- **Containerization**: Docker

## GitHub Repository

The source code for CosmicVanilla Web can be found in its GitHub repository: [hycord/cosmicvanilla-web](https://github.com/hycord/cosmicvanilla-web).
